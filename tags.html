HTML CheatSheet

This page lists HTML elements learned so far in codecademy, seen in the table below as they would appear in their tags.

Element Notes
<!DOCTYPE html> Always include at the top so browser knows how to interpret your work.
<html> Root element. Allows you to specify a language attribute. Contains all HTML code.
<head> Contains metadata, including title and links to stylesheets and scripts.
<title> Include in head element. Appears in browser tab.
<a> Link to internal or external pages, or content on the same page using #id element. Literally anchor.
<!--comment--> Make notes within your code that won't show up on the actual page.
<body> Main content goes here.
<h1> - <h6> Six levels of section headings.
<nav> Represents a navigation section.
<blockquote> For a longer quotation. Usually indented. Use cite attribute, or cite element if you want the citation to appear on the page.
<div> Sets up a generic container for easier styling.
<ol> Ordered (numbered) list.
<ul> Unordered (bulleted) list.
<li> An item in an ordered or unordered list.
<abbr> Abbreviation or acronym. Use title attribute to define the initials or abbreviation.
<br> Line break or carriage return.
<cite> Cite the source you've referenced in your content.
<code> Indicates that the text is a piece of code, usually rendered in monospace.
<li> An item in an ordered or unordered list.
<em> Used to emphasize or stress a word or phrase.
<q> Use for short, inline quotations.
<s> Strikethrough.
<span> Essentially the inline version of div. Div is block, Span is inline.
<strong> Indicates importance or urgency, usually rendered in bold.
<sub> ASubscript.
<audio> Embed sound. Use src attribute.
<img> Embed an image. Use src attribute.
<video> Embed a video. Use src attribute.
<del> Shows a piece of text has been deleted. Rendered as strikethrough.
<ins> Indicates that text has been inserted.
<table> Indicates table.
<thead> Indicates the table header row.
<tr> Indicates a table row.
<th> Indicates a cell that contains a table header.
<td> Indicates a cell that contains table data.
<tbody> Indicates the section of the table that contains the main data. Not header or footer.
<tfoot> Indicates the footer section of the table.
<button> Creates a clickable button.
<input> Allows user to interact with a form with different types of data depending on the input attribute.
