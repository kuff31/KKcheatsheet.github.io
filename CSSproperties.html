CSS: Properties

This page lists and describes CSS properties.

Property Notes
background-color Lets you color the background of the specified element.
background-image Lets you pick an image for the background instead of just a color.
border-radius Use this to make rounded corners.
border Syntax: weight style color. You can also specify to border-top, border-right, etc.
bottom Set the element's position from the bottom.
clear What happens with the element next to a floating element.
color Sets the text color.
cursor Specify the type of cursor used when pointing to an element.
display How to display an element. Can be inline, block, or inline-block.
float Specifies whether an element floats left, right, or none.
font-family Sets the kind of font. Can be as specific as Arial or as vague as sans-serif.
font-size Sets the size of text. Px, ems.
font-style Normal, italic, oblique.
font-weight Normal/400, bold/700, bolder, lighter, 100 through 900.
height Sets the height. Can be different units.
left Sets the position from the left.
line-height Sets the line height.
list-style For <ul> and <ol>. Syntax: type position image.
margin Sets the margin properties. Use 0 auto to center horizontally.
opacity Sets the opacity for an element with a value from 0 to 1.
overflow Choose what happens to content that flows outside the element's box.
padding Sets the amount of space between content and border.
position Can be static (default), relative, absolute, or fixed.
right Sets position from the right.
text-align Sets the horizontal alignment: left, center, right, justify.
text-decoration Sets to "none" to get rid of underline on links.
text-transform Uppercase or lowercase all letters in specified element.
top Sets the position from the top.
transition-duration How long an effect like the :hover effects take.
vertical-align Sets the vertical alignment of an element.
width Sets the element's width.
word-spacing Sets the space between words.
z-index Can position things behind or in front of one another.
